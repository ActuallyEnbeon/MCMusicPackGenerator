# MCMusicPackGenerator
A tool to simplify adding new music tracks to Minecraft: Java Edition.

This tool automates the process of creating resource packs that add new music.

## Features
* Add tracks
* Customise track information, including adjusting volume
* Make tracks play for any vanilla Minecraft music events
* Customise pack name, description, and icon
* Supports all Minecraft: Java Edition versions since 1.6.1
* Advanced mode:
    * Manually control a pack's supported pack formats
    * Adjust the weight (likelihood) of tracks playing per-event
    * Add custom sound events, including modded events

## How to use
1. Clone the repository.
2. Open up `index.html` in your favourite browser.
3. Start creating a pack!
